<task type="auto" tdd="true">
  <name>Task 1: Token store persistence and JWT auth service</name>
  <files>internal/server/store/tokens.go, internal/server/store/tokens_test.go, internal/server/auth/jwt.go, internal/server/auth/jwt_test.go</files>
  <behavior>
    - TokenStore.RevokeToken(jti, userID, expiresAt) inserts into revoked_tokens table
    - TokenStore.LoadActiveRevocations() returns all non-expired revocations
    - TokenStore.CleanExpired(now) deletes rows where expires_at < now
    - Service.IssueToken(user) returns a signed HS256 JWT with claims: jti (UUID), sub (user_id), uid, email, role, iss ("claude-plane"), iat, exp
    - Service.ValidateToken(tokenString) returns claims for valid tokens
    - ValidateToken rejects: expired tokens, wrong signing method (alg:none attack), invalid signature, revoked JTI
    - Service.RevokeToken(jti, expiresAt) adds to blocklist (in-memory + DB)
  </behavior>
  <action>
1. Create `internal/server/store/tokens.go`:
   - `revoked_tokens` table creation SQL (jti TEXT PK, user_id TEXT NOT NULL, revoked_at DATETIME DEFAULT CURRENT_TIMESTAMP, expires_at DATETIME NOT NULL) with index on expires_at. Add this as an additional migration call in the store initialization or as a method that runs alongside existing migrations.
   - `RevokeToken(jti, userID string, expiresAt time.Time) error` — INSERT into revoked_tokens using writer pool
   - `LoadActiveRevocations() ([]RevokedToken, error)` — SELECT where expires_at > now from reader pool
   - `CleanExpired(now time.Time) error` — DELETE where expires_at <= now using writer pool
   - `RevokedToken` struct: JTI, UserID, RevokedAt, ExpiresAt

2. Create `internal/server/auth/jwt.go`:
   - `Claims` struct embedding `jwt.RegisteredClaims` with UserID, Email, Role fields
   - `Service` struct with signingKey []byte, tokenTTL time.Duration, blocklist *Blocklist
   - `NewService(signingKey []byte, tokenTTL time.Duration, blocklist *Blocklist) *Service`
   - `IssueToken(user *store.User) (string, error)` — create claims with uuid.New() for JTI, sign with HS256
   - `ValidateToken(tokenString string) (*Claims, error)` — parse with jwt.WithValidMethods([]string{"HS256"}) and jwt.WithIssuer("claude-plane"), check blocklist
   - `RevokeToken(jti string, expiresAt time.Time) error` — delegate to blocklist.Add()

3. Install dependencies: `go get github.com/golang-jwt/jwt/v5@latest github.com/google/uuid@latest`

4. Write tests first (TDD):
