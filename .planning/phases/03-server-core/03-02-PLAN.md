   - `(s *Store) UpsertMachine(machineID string, maxSessions int32) error` — INSERT OR REPLACE into machines table using writer pool. Set status="disconnected", created_at=CURRENT_TIMESTAMP on insert, updated_at=CURRENT_TIMESTAMP always.

   - `(s *Store) GetMachine(machineID string) (*Machine, error)` — SELECT from reader pool, return nil and sql.ErrNoRows-based error if not found

<task type="auto" tdd="true">
  <name>Task 2: Connection manager with DB-backed status tracking</name>
  <files>internal/server/connmgr/manager.go, internal/server/connmgr/manager_test.go</files>
  <behavior>
    - Register(machineID, agent) adds agent to in-memory map and updates DB to "connected"
    - Register with existing machineID cancels old connection and replaces
    - Disconnect(machineID) removes from map and updates DB to "disconnected"
    - GetAgent(machineID) returns ConnectedAgent or nil
    - ListAgents() returns slice of AgentInfo for all connected agents
    - All operations are thread-safe under concurrent access
  </behavior>
  <action>
1. Create `internal/server/connmgr/manager.go`:
   - `ConnectedAgent` struct: MachineID string, RegisteredAt time.Time, MaxSessions int32, Cancel context.CancelFunc (the stream field is typed as `interface{}` for now to avoid importing proto package — will be typed properly when gRPC wiring happens in Phase 4)
   - `AgentInfo` struct: MachineID, Status string, MaxSessions int32, ConnectedAt time.Time (public DTO for REST API)
   - `MachineStore` interface: `UpsertMachine(machineID string, maxSessions int32) error`, `UpdateMachineStatus(machineID, status string, lastSeenAt time.Time) error` — allows testing with a mock store
   - `ConnectionManager` struct: sync.RWMutex, agents map[string]*ConnectedAgent, store MachineStore, logger *slog.Logger
   - `NewConnectionManager(store MachineStore, logger *slog.Logger) *ConnectionManager`
   - `Register(machineID string, agent *ConnectedAgent) error` — Lock, if old exists call old.Cancel(), add to map, Unlock, call store.UpsertMachine then store.UpdateMachineStatus("connected", time.Now())
   - `Disconnect(machineID string)` — Lock, delete from map, Unlock, call store.UpdateMachineStatus("disconnected", time.Now())
   - `GetAgent(machineID string) *ConnectedAgent` — RLock, return agent or nil, RUnlock
   - `ListAgents() []AgentInfo` — RLock, iterate map, build slice, RUnlock

2. Write tests first (TDD), use a mock MachineStore:
   - TestRegisterAgent: register, verify GetAgent returns it, verify ListAgents includes it
   - TestRegisterReplaces: register agent A, register agent B with same machineID, verify old cancel was called, new agent returned
   - TestDisconnect: register then disconnect, verify GetAgent returns nil, ListAgents is empty
   - TestListMultipleAgents: register 3 agents, verify ListAgents returns all 3
   - TestConcurrentAccess: launch 50 goroutines doing Register/Disconnect/ListAgents/GetAgent, run with -race to verify no data races
   - Mock store: simple struct that records calls (UpsertMachine, UpdateMachineStatus) for assertions
  </action>

  </verify>
