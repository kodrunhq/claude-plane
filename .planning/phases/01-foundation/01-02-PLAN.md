  <files>internal/shared/tlsutil/ca.go, internal/shared/tlsutil/ca_test.go, internal/shared/tlsutil/loader.go</files>

    - IssueServerCert: creates server.pem and server-key.pem signed by CA; contains SANs (localhost, 127.0.0.1, and provided hostnames); 2yr validity; ExtKeyUsage=ServerAuth
    - IssueAgentCert: creates agent.pem and agent-key.pem signed by CA; CN=machine-id; 2yr validity; ExtKeyUsage=ClientAuth
    - Random 128-bit serial numbers on all certs (not hardcoded)
    - mTLS handshake test: generate CA + server cert + agent cert in temp dir, create tls.Config for both sides, do a TLS handshake — must succeed

    - AgentTLSConfig: loads CA cert, agent cert+key, returns *tls.Config for dialing server
  </behavior>
  <action>
1. Create `internal/shared/tlsutil/ca.go`:
   - `GenerateCA(outDir string) error` — ECDSA P-256 key, self-signed CA cert, 10yr validity, random serial, writes PEM files
   - `IssueServerCert(caDir, outDir string, hostnames []string) error` — loads CA, generates server leaf cert with SANs, ExtKeyUsage ServerAuth, 2yr validity, random serial
   - `IssueAgentCert(caDir, outDir, machineID string) error` — loads CA, generates agent leaf cert with CN=machineID, ExtKeyUsage ClientAuth, 2yr validity, random serial
   - Helper: `loadCA(caDir string) (*x509.Certificate, *ecdsa.PrivateKey, error)` — reads ca.pem and ca-key.pem
   - Helper: `writePEMFiles(dir, certName, keyName string, certDER []byte, key *ecdsa.PrivateKey) error`

2. Create `internal/shared/tlsutil/loader.go`:
   - `ServerTLSConfig(caCertPath, serverCertPath, serverKeyPath string) (*tls.Config, error)` — loads CA into cert pool, loads server keypair, sets ClientAuth=RequireAndVerifyClientCert
   - `AgentTLSConfig(caCertPath, agentCertPath, agentKeyPath string) (*tls.Config, error)` — loads CA into RootCAs, loads agent keypair, returns config for tls.Dial

3. Write tests first (TDD), then implement. Tests use t.TempDir() for all file I/O.
